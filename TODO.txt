
CLI VERSION : 
 pom.xml vs Version.java + .bat / .sh 
 
GENERAL COMMANDS :
  'cfg' : print configuration ( telosys-tools.cfg )
  'var' or 'lv' : print variables
  
DB MODEL COMMANDS : (NB : "db" is for "Delete Bundle" )

(X) ldb : list all databases ( show the selected db, eg : "1 [2] 3 4" if '2' is the selected db )

cdb : check the current (or given) db  
 cdb [1] --> print catalogs and schemas (par defaut)
 cdb [1] -i --> info   
 cdb [1] -t --> tables  
 cdb [1] -c --> columns
 cdb [1] -pk --> primary keys
 cdb [1] -fk --> foreign keys
 cdb [1] -cat --> catalogs
 cdb [1] -sch/s --> schemas

(X) ndbm : new db model
  (X) ndbm   (for default/selected db)
  (X) ndbm 2 ( new db model from database #2 )

udbm : update db model ( ou "udbmodel" )
 udbm
 udbm 1 ( update db model from database #1 )


NEW COMMANDS ( IN THE FUTURE ):

cb : check bundle

LAUNCHERS COMMANDS : 
 'nl' : new launcher
 'll' : list launchers
 'el' : edit launchers
 'launch'/'run' : launch a launcher  or  'gen < launcher-name'
